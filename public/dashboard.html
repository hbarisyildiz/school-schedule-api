📡 API Endpoints

POST /api/auth/login - Kullanıcı girişi
POST /api/auth/register - Yeni okul kaydı
GET /api/user - Kullanıcı bilgileri (token)
GET /api/test - API test endpoint (token)
GET /api/my-school - Okul bilgileri (token)
GET /api/subscription-plans - Abonelik planları
GET /api/schools - Tüm okullar (super admin)
